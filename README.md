# GDX Landing

How to run project:

### Development mode

Install all packages, using command:

```bash
  npm install
```

Then, run project, using this command:

```bash
  npm run dev
```

### Production mode

Build project, with this command:

```bash
  npm run build
```

Next, boot up a local web server that serves the files from dist:

```bash
  npm run preview
```
